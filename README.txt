How to Run This Local Portfolio Web

1. Extract UTS_Rizky_Maulana to C:/xampp/htdocs/portfolio.
2. Make a database named "portodata" in localhost/phpmyadmin then import the sql file from this directory.
3. Change the username and password in koneksi.php based on your own phpMyAdmin.
4. To go to admin page, go to the footer the page below.
